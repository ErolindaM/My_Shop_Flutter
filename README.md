# Grocery Store App (Flutter)

## Overview
This is a beginner-level Flutter project that demonstrates a simple grocery store mobile application.  
Users can browse grocery products and add them to a shopping cart for purchase.

The project is designed for learning purposes and focuses on basic Flutter concepts and UI development.

##  Features
- List of grocery products  
- Add products to cart  
- Remove products from cart  
- View cart items  
- Simple and clean UI  

## Technologies Used
- Flutter  
- Dart  
- Material Design
